# Collision Calendar — Arc One, Day 1

> Visual overlay of all streams competing for the player's time and attention.
> The guaranteed-slip principle: in every time slot, the player has more
> compelling options than they can pursue. Something always falls behind.

---

## Timeline Overview

```
MORNING         MIDDAY              AFTERNOON         EVENING           NIGHT
─────────────── ─────────────────── ───────────────── ───────────────── ─────
Arrive → Dorm → [Admin OR Lunch] → [Glenn + other] → Evening Choice → Sleep
```

## Guaranteed-Slip Summary

| Slot | What slips | Why it matters |
|------|-----------|---------------|
| Dorm (timing choice) | Emotional context of Glenn encounter | Can't optimize Glenn because you don't know he exists |
| Admin (if rushed) | Elective browsing, CS listing | Glenn's "computers" line has no anchor |
| Glenn bench (if before lunch) | On-time lunch arrival | Late arrival = enter conversation mid-flow |
| Lunch (Doug/Keith moment) | Alliance leaning | Laughing with Doug costs Keith |
| Evening | Two of three activities | The world moves without you |

---

## Path Comparison

### Path A: Admin Before Lunch
**Rhythm:** Efficient → rushed → destabilized → social (but late) → choice → rest
**Glenn context:** Time-pressured. The bench costs something.

### Path B: Lunch First
**Rhythm:** Social → bonded → bureaucratic → destabilized (with full attention) → choice → rest
**Glenn context:** Relaxed. Full capacity to absorb.

### Path C: Solo / Noncommittal
**Rhythm:** Withdrawn → alone → destabilized (raw) → isolated → choice → rest
**Glenn context:** No social bonds yet. "You're going to need people" is an open wound.

## Anti-Optimization Audit

| Can the player... | Answer |
|-------------------|--------|
| Browse electives AND arrive at lunch on time? | **No** |
| Sit with Glenn as long as they want AND keep lunch commitment? | **No** |
| Attend the party AND the card game? | **No** |
| Bond with Doug AND not alienate Keith at lunch? | **No** |
| Go to the union AND build social capital? | **No** |
| Meet Glenn AND avoid destabilization? | **No** |

**Verdict:** Something always slips. The principle holds.
